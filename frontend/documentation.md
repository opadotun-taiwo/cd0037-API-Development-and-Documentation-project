# API Reference
## Introduction
It is a question and answer web app that udacity students can use to brush up their memory on different Knowledge category

## Base URl
At present this app is not hosted and only runs locally. http://127.0.0.1:5000/

## Error handling
Errors are returned in a JSON format
```
{
    'success':False,
    'error':404,
    'message': "resourse not found"
}
```
The API will return four error type when reuest fail:
404 : resourse not found
405 : method not allowed
422 : request cannot be processed
500 : An internl server error occured

## End points
**GET/categories**
Description:
    Returns list of categories
Sample: e.g curl http://127.0.0.1:5000/categories

Response:
```
{
  "1": "Science",
  "2": "Art",
  "3": "Geography",
  "4": "History",
  "5": "Entertainment",
  "6": "Sports"
}
```
